# Privacy Policy

This repository contains a simple static privacy policy page for the "Snake Game" created by Grillin_and_chillin.

Files

- `index.html` — the privacy policy page you can link to directly.

Usage

- View the page on GitHub by opening `index.html`.
- To host the page on GitHub Pages, enable Pages for this repository (Settings → Pages) and select the `main` branch, then the site will be available at `https://<your-username>.github.io/Privacy-policy/`.
